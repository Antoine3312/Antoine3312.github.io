Capteur MQTT

Ce projet était en lien avec le tableau de bord Java. Le but de ce programme python était de s'abonner un flux MQTT et de récupérer les messages transmis sur ce dernier.

Les émetteurs devaient simuler des émetteurs situer dans les entrepôts de l'entreprise. Pour cela, elle récupère les données situé dans les salles de cours.

Ce projet a été pour moi l'occasion de développer mes connaissances en Python et notamment sur la bibliothèque paho-mqtt.
